2020

InfraFix

2021

O projeto InfraFix foi criado com o objetivo de desenvolver uma aplicação que facilite a identificação de infraestruturas danificadas em uma região. A aplicação permite que os usuários capturem fotografias dessas infraestruturas usando seus dispositivos móveis. Essas imagens são então enviadas para um servidor, onde são armazenadas em uma base de dados.
